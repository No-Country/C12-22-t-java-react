# Welcome appStore!
## Descripcion del Proyecto:
Es una plataforma tienda online con productos de apple. el cual permite a los clientes relizar compras de productos de apple en linea conociendo antes los productos disponibles.

## STACK TECNOLOGICO
### FRONTEND
- ReatJS
### BACKEND
- Java 11
- Springboot
- JPA
- Lombok
- Spring Security
- Swagger
- -h2 Data base
